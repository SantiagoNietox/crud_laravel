PASOS PARA PONER A FUNCIONAR EL PROYECTO

1. PARA QUE ESTE PROYECTO PUEDA FUNCIONAR DEBERAS IMPORTAR LA BASE DE DATOS EN TU SERVIDOR LOCAL, LA BD LA PODRAS ENCONTRAR EN UNA CARPETA DE ESTE PROYECTO LLAMADA BASE DE DATOS.

2. UNA VEZ IMPORTADA LA BASE DE DATOS EN TU SERVIDOR LOCAL LO QUE TIENES QUE HACER ES INICIAR EL SERVIDOR.

3. UNA VEZ EL SERVIDOR INICIADO, YA PODRAS ENTRAR A EL PORYECTO, NO TE OLVIDES DE PONER LA CARPETA DEL PORYECTO YA SEA EN TU HTDOCS O EN TU WWW, DEPENDE DE EL SERVICIO QUE UTILICES.

4. UNA VEZ EL PROYECTO PUESTO EN FUNCIONAMIENTO ENCONTRARAS DOS BOTONES, UNO PARA REGISTRARTE Y OTRO PARA VER A LOS USUARIOS REGISTRADOS.
